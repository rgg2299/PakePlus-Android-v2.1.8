# 🎮 输入法按键编辑器 - 快速使用指南

## 🚀 快速开始

### 打开编辑器
双击 `index.html` 文件，在浏览器中打开编辑器。

### 布局说明
编辑器采用三栏布局：
- **左侧面板**：基本设置和预设模板
- **中间面板**：实时预览和操作按钮
- **右侧面板**：样式设置（填充、描边、阴影、光晕）

---

## ✨ 核心功能

### 1️⃣ 预设模板
点击顶部的预设按钮，快速应用不同的按键样式：
- 🎨 默认、现代、玻璃效果
- ✨ 霓虹、极简、复古风格
- 🌈 多种渐变和材质效果

### 2️⃣ 文字控制
- **输入文字**：在"按键文字"框中输入任意汉字或字母
- **清空文字**：清空输入框后，预览框文字自动消失，方便查看纯样式
- **调整大小**：使用滑块调整文字大小（12-80px）
- **更改颜色**：点击颜色选择器更改文字颜色

### 3️⃣ 基本尺寸
- **宽度/高度**：调整按键的尺寸（40-300px）
- **圆角半径**：从方形到圆形任意调整
- **边缘留白**：设置文字与边框的距离
- **图边留白**：设置导出图片时的边距
- **整体透明度**：调整按键的透明度（0-100%）

### 4️⃣ 填充颜色（右侧面板）
- **纯色填充**：选择单一背景色
- **渐变填充**：设置起始色、结束色和角度
- **实时预览**：渐变预览条显示当前效果

### 5️⃣ 描边设置（右侧面板）
- **启用/禁用**：勾选复选框启用描边
- **描边宽度**：1-20px
- **描边颜色**：自定义颜色
- **描边类型**：实线、虚线、点线
- **描边位置**：外描边 / 内描边

### 6️⃣ 阴影设置（右侧面板）
- **启用/禁用**：勾选复选框启用阴影
- **偏移调整**：水平和垂直偏移（-30到30px）
- **模糊程度**：0-50px
- **阴影颜色**：自定义颜色
- **阴影类型**：外阴影 / 内阴影

### 7️⃣ 光晕效果（右侧面板）
- **启用/禁用**：勾选复选框启用光晕
- **光晕大小**：0-100px
- **光晕强度**：0-50px
- **光晕颜色**：自定义颜色
- **光晕透明度**：0-100%
- **光晕类型**：外光晕 / 内光晕

---

## 🖱️ 高级功能

### 自由拖动预览框
1. 将鼠标移动到预览框上
2. 按住鼠标左键并拖动
3. 预览框可以移动到任意位置
4. 拖动后自动禁用滚动同步功能

### 智能滚动同步
滚动左侧设置面板时，预览框会根据当前区域自动移动：
- 📐 基本设置 → 居中显示
- 🎨 填充颜色 → 向上偏移
- ✏️ 描边设置 → 向右偏移
- 🌈 阴影设置 → 向下偏移
- ✨ 光晕效果 → 向左偏移

**注意：** 手动拖动预览框后，滚动同步会自动禁用

---

## 🔘 按钮功能

### 🔄 重置
将所有设置恢复到默认值（"默认"预设）

### 📍 重置位置
将预览框恢复到中心位置，并重新启用滚动同步功能

### 💾 下载图片
将当前按键导出为 PNG 图片，文件名格式：`key-[文字]-时间戳.png`

## 💡 使用技巧

### 技巧1：快速预览多种样式
1. 点击不同的预设按钮
2. 观察右侧预览框的实时变化
3. 找到喜欢的样式后，再进行微调

### 技巧2：专注于某个属性
1. 滚动到对应的设置区域
2. 观察预览框的自动移动，帮助定位
3. 调整参数，查看实时效果

### 技巧3：查看纯样式效果
1. 清空"按键文字"输入框
2. 观察背景、边框、阴影等纯样式
3. 适合检查按键的整体设计

### 技巧4：自由调整预览框位置
1. 将预览框拖动到合适位置（如屏幕角落）
2. 这样在调整设置时不会遮挡视线
3. 如需恢复自动同步，点击"📍 重置位置"

### 技巧5：创建对称设计
1. 先设计一个按键
2. 记录各项参数
3. 创建另一个按键，使用相同参数但更换文字

## 🎨 常见应用场景

### 场景1：创建简洁的字母按键
1. 点击"极简"预设
2. 输入字母（如 "A"、"B"、"C"）
3. 调整文字大小和颜色
4. 下载图片

### 场景2：设计漂亮的汉字按键
1. 点击"玻璃"预设
2. 输入汉字（如 "我"、"你"、"他"）
3. 启用光晕效果，增加发光感
4. 调整透明度，创造半透明效果

### 场景3：制作霓虹风格按键
1. 点击"霓虹"预设
2. 输入文字
3. 增加光晕大小和强度
4. 调整阴影，增强立体感

### 场景4：创建复古材质按键
1. 点击"木质"或"皮革"预设
2. 使用渐变颜色模拟材质纹理
3. 增加内阴影，增加深度感
4. 调整描边，强化边缘

## ⚠️ 注意事项

1. **浏览器要求**：推荐使用 Chrome、Firefox、Edge 或 Safari 最新版

2. **图片格式**：目前只支持导出 PNG 格式，支持透明背景

3. **文字支持**：支持汉字、字母、数字和常用符号

4. **性能建议**：如果预览出现卡顿，可以关闭某些效果（如光晕、阴影）

5. **保存建议**：下载的图片建议及时重命名，避免文件名重复

## 🆘 故障排除

### 问题：预览框不显示
- 检查浏览器控制台是否有错误（按 F12）
- 刷新页面重试

### 问题：拖动后找不到了
- 点击"📍 重置位置"按钮
- 预览框会回到中心位置

### 问题：下载的图片不完整
- 增加"图边留白"参数
- 确保阴影和光晕效果在图片范围内

### 问题：样式没有变化
- 检查对应功能的复选框是否勾选
- 确认数值是否在有效范围内
