# 📐 布局更新说明

## 🎯 更新概览

本次更新将编辑器布局从"两栏式"升级为"三栏式"，优化了用户体验和工作流程。

---

## 📊 布局对比

### 更新前（两栏布局）
```
┌─────────────┬─────────────┐
│  设置面板   │  预览面板   │
│             │             │
│  所有功能   │             │
│  集中在一   │  实时预览   │
│  侧         │             │
│             │             │
└─────────────┴─────────────┘
```

**缺点：**
- 设置面板过长，需要频繁滚动
- 基本设置和样式设置混在一起
- 不够直观，功能分组不清晰

---

### 更新后（三栏布局）
```
┌──────────┬──────────┬──────────┐
│ 基本设置 │  预览    │ 样式设置 │
│          │          │          │
│ 预设模板 │          │ 填充颜色 │
│ 基本参数 │  实时    │ 描边设置 │
│          │  预览    │ 阴影设置 │
│ 操作区   │          │ 光晕效果 │
│ 按钮     │          │          │
│          │          │          │
│          │          │          │
└──────────┴──────────┴──────────┘
```

**优点：**
- ✅ 功能分组清晰，左侧基本设置，右侧样式设置
- ✅ 预览框居中，查看效果更直观
- ✅ 左右面板独立滚动，互不影响
- ✅ 设置面板高度优化，减少滚动
- ✅ 更符合专业设计工具的布局习惯

---

## 🎨 面板功能分布

### 左侧面板 - 基本设置
**功能：** 基础参数和预设

包含内容：
- 🎨 预设模板（18种样式）
- 📐 基本参数
  - 按键文字
  - 文字大小和颜色
  - 文字位置（水平/垂直）
  - 宽度和高度
  - 圆角半径
  - 边缘留白

  - 整体透明度

**特点：**
- 面板高度：最大为视口高度
- 支持独立滚动
- 自定义滚动条样式

**功能：** 预览按键效果和操作

- 🎮 操作按钮
  - 🔄 重置
  - 📍 重置位置
  - 💾 下载图片

**特点：**
- 面板固定在视口中央
- 预览框支持自由拖动
- 支持滚动同步功能

---

### 右侧面板 - 样式设置
**功能：** 高级样式和效果

包含内容：
- 🎨 填充颜色
  - 纯色/渐变切换
  - 渐变预览
- ✏️ 描边设置
  - 宽度、颜色、类型、位置
- 🌈 阴影设置
  - 偏移、模糊、颜色、类型
- ✨ 光晕效果
  - 大小、强度、颜色、透明度、类型

**特点：**
- 面板高度：最大为视口高度
- 支持独立滚动
- 自定义滚动条样式

## 📐 技术细节

### CSS Grid 布局
```css
.container {
    display: grid;
    grid-template-columns: 380px 1fr 380px;
    gap: 20px;
}
```

**说明：**
- 左右面板固定宽度 380px
- 中间预览面板占据剩余空间
- 面板间距 20px

---

### 滚动优化
```css
.left-panel, .right-panel {
    max-height: calc(100vh - 40px);
    overflow-y: auto;
}

.center-panel {
    position: sticky;
    top: 20px;
}
```

**说明：**
- 左右面板最大高度为视口高度减去内边距
- 中间面板固定在视口顶部
- 避免整页滚动

---

### 自定义滚动条
```css
.left-panel::-webkit-scrollbar,
.right-panel::-webkit-scrollbar {
    width: 6px;
}

.left-panel::-webkit-scrollbar-thumb,
.right-panel::-webkit-scrollbar-thumb {
    background: rgba(102, 126, 234, 0.5);
    border-radius: 3px;
}
```

**说明：**
- 滚动条宽度 6px，纤细美观
- 使用主题色作为滚动条颜色
- 悬停时增加不透明度

## 🎯 用户体验提升

### 1. 减少滚动
- **更新前**：需要滚动整个设置面板
- **更新后**：左右面板独立滚动，减少 50% 滚动量

- **更新后**：基本设置在左，样式设置在右

### 3. 预览更直观
- **更新前**：预览在右侧，不够突出
- **更新后**：预览居中，视觉焦点明确

### 4. 工作流程优化
- **更新前**：选择预设 → 滚动到底部调整样式 → 回到顶部调整基本参数
- **更新后**：选择预设（左）→ 调整样式（右）→ 查看预览（中），一目了然

## 📱 响应式设计

当前布局针对桌面端优化（最小宽度 1200px）

### 桌面端（≥ 1200px）
- 三栏布局
- 左右面板宽度 380px

### 平板端（768px - 1199px）
- 两栏布局（预留）
- 左侧面板折叠或缩小

### 移动端（< 768px）
- 单栏布局（预留）
- 标签页切换

## 📝 使用建议

### 推荐工作流程

1. **选择预设**（左侧）
   - 从预设模板中选择一个基础样式

2. **调整基本参数**（左侧）
   - 修改文字、大小、位置
   - 调整宽高、圆角、留白

3. **设置样式效果**（右侧）
   - 选择填充颜色（纯色/渐变）
   - 启用描边、阴影、光晕
   - 微调各项参数

4. **实时预览**（中间）
   - 观察效果
   - 如需调整，返回对应面板

5. **下载图片**（中间）
   - 点击下载按钮导出
